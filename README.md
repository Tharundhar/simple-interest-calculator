# github-final-project

## Simple Interest Calculator

## Project Overview
This project provides a simple Bash-based calculator to compute **Simple Interest** based on user-provided inputs.

## Description
Simple Interest is calculated using the formula:

```
Simple Interest = (Principal × Rate × Time) / 100
```

## Features
- Accepts user input for Principal, Rate of Interest, and Time Period
- Calculates and displays the Simple Interest
- Lightweight Bash script — no dependencies required

## Usage
```bash
bash simple-interest.sh
```

## Inputs
| Field              | Description                          |
|--------------------|--------------------------------------|
| Principal          | The initial amount of money          |
| Rate of Interest   | Annual interest rate (in percentage) |
| Time Period        | Duration in years                    |
